# مرشد الحرم الذكي

تطبيق ويب تقدمي PWA تجريبي للمرشدين داخل المسجد الحرام.

## الميزات

- واجهة عربية قابلة للتثبيت على iPhone و Android عبر المتصفح.
- اختيار موقع المرشد ووجهة الضيف.
- حساب مسار إرشادي من خلال Graph داخلي.
- دعم لغات: العربية، الإنجليزية، التركية، الأوردية، الفارسية، الإندونيسية، الفرنسية، البنغالية.
- توليد إرشادات نصية وصوتية عبر Web Speech API عندما يدعم المتصفح ذلك.
- يعمل دون اتصال بعد أول تحميل بفضل Service Worker.

## ملاحظة مهمة

البيانات الحالية تقريبية لأغراض التجربة فقط، وليست بيانات رسمية للحرم. يجب استبدال `data.js` ببيانات معتمدة من خريطة الحرمين عند توفرها.

## التشغيل المحلي

افتح `index.html` مباشرة أو شغّل خادماً محلياً:

```bash
python3 -m http.server 8080
```

ثم افتح:

```text
http://localhost:8080
```

## التثبيت على iPhone

افتح الرابط في Safari ثم اضغط زر المشاركة واختر **Add to Home Screen**.
